# aol-database

Link Dokumentasi: https://docs.google.com/document/d/1jZ0B4VNyw0uJh-SmhtDkSnceMxdASs0B_AruixK4daM/edit?tab=t.0

Langkah-langkah membuat database sesuai kriteria soal:
1. Jalankan perintah SQL dari schema.sql di phpmyadmin menggunakan XAMPP
2. Unzip cleanedcsv.zip
3. Jalankan perintah di import.txt di command prompt
4. Jalankan perintah SQL dari queries.sql di phpmyadmin menggunakan XAMPP
5. Jalankan perintah SQL dari advanced.sql di phpmyadmin menggunakan XAMPP
